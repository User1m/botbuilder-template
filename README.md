# BotBuilder Template

Sample template for building bots using Microsoft Bot Framework

## Concepts
* A Skill is a mapping of Intents to Dialogs
 * When I say 'Login' I want the 'LoginDialog'

 

## Setup
1. Clone repo
2. Install packaages
3. Build project
4. Start server

### Clone

```
git clone git@github.com:User1m/botbuilder-template.git

```

### Installing

```
npm i

```

### Building

```
npm run build

```

### Running

```
npm run start

```

### Running tests

```
npm test
```
